# bnf: Birkhoff normal form

Compute the Birkhoff normal form of a Hamiltonian.

## Features

- Arbitrary order
- Lightweight and easy to use.

## Installation

Install this module with pip

```sh
pip install bnf
```

## Quickstart

An example
```python
from bnf import *
```
